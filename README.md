# jump-to-flask
[Jump to flask](https://wikidocs.net/book/4542) 의 내용을 바탕으로 공부하면서 작성한 내용입니다.

간단한 딥러닝 모델을 서빙을 목적으로 공부하고 있습니다.

목차의 링크는 두가지로 나누어져 있는데 목차의 명이 적혀 있는곳을 누르면 해당하는 목차의 내용을 가진 디렉토리로 이동합니다.
